Execute Tests Guideline
=====

**From terminal:**
1. Open a terminal.
2. Go to solution folder.
3. Run ```dotnet test```
4. Wait for the results.


**From IDE (JetBrains Rider Example):**
1. Open the solution in **JetBrains Rider**.
2. Right-click on the **Tests** project.
3. Select **Run Unit Tests** or similar.
4. View the result in the **Tests** window.
